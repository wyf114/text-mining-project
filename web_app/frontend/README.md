# web_app

## Frontend setup --------------------------------------------------
```
cd web_app\frontend

npm install
```
### Compiles and hot-reloads for development
```
npm run serve
```
### Compiles and minifies for production
```
npm run build
```
### Lints and fixes files
```
npm run lint
```

### Backend setup --------------------------------------------------
```
cd web_app\backend

python api.py
```

### Install dependencies
```
frontend:
npm install axios

backend:
pip install nltk
pip install num2words
pip install networkx
```
